# Initialization

## Permanent Portfolio

永久投资组合配置 4 种资产：

- 25% 股票
- 25% 债券
- 25% 现金
- 25% 黄金

通过分散化资产类别应对不同经济状况。被动管理而非积极管理。

### 4 种经济状况

1. **繁荣**：生产力与收益提高，低失业率，稳定或下跌的利率。对经济状况持乐观态度，股票走势良好。债券表现也不错，但是不如股票。不利好没有利息与分红的黄金。
2. **通货紧缩**：价格下降，利率下跌，货币价值上涨。利率下跌利好长期债券。由于现金购买力增加，现金类资产表现良好。价格下降使公司利润下跌，股票表现糟糕。而黄金通常表现不太好。
3. **衰退**：在永久投资组合的语境下，衰退指的是银根紧缩衰退期。在银根缩紧衰退期，央行提高利率来降低高通胀，带来的不良后果就是经济衰退。股票与债券表现不佳。黄金通常表现也不太好。此时现金可以用来购入价格较低的其他资产。
4. **通货膨胀**：货币价值下跌，价格上涨，伴随利率上涨。高通胀使资金贬值，黄金可以对资产加以保护。
